# Data Migration

Rocket.Chat lets you import all your data if you are moving from another service like Slack to Rocket.Chat

To access this menu, navigate to **Administration** > **Workspace** > **Import**.

To Import your data,

* Click **Import New File.**
* Select **Import Type.**
